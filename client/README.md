# 프론트엔드

소유자: 유상욱
태그: 코드베이스

## 패키지

- nodejs-nextjs
    
    node.js 기반 SSR 프레임워크
    
- Tailwind CSS
유틸리티 클래스 CSS 프레임워크
- Headless UI
Unstyled CSS 프레임워크
- Heroicons
오픈소스 아이콘 라이브러리
- Firebase
npm 파이어베이스 패키지
